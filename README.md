Fitness Tracker Dashboard

A web-based fitness tracking dashboard that allows users to manage workouts, track weekly progress, and visualize fitness activity through interactive charts all in a clean, modern interface.
This project was built to demonstrate front-end development skills using HTML, CSS, and JavaScript, with a focus on UI structure, user interaction, and data visualization.


Purpose and Problem It Solves

Many beginner fitness trackers are either too complex or lack meaningful visual feedback.  
This project aims to provide:
- A simple way to log workouts
- Visual motivation through progress rings and charts
- A personalized experience using gender-based profile selection
- A dashboard-style layout similar to real-world fitness applications
- The goal is to help users stay consistent and motivated while keeping the interface intuitive and easy to use.

 Key Features
 User Profile with Gender Selection**
 - Users select a gender profile which is reflected across the dashboard using avatars.

 Workout Logging
  - Add workouts with type, duration, and calories burned.
  - Select workout types using interactive cards.

  Weekly Progress Tracking
  - Progress rings show weekly workout completion.
  - Charts visualize workout consistency and streaks.

  Dark Mode
  - Toggle between light and dark themes for better accessibility and user comfort.

  Persistent Data
  - Workout history and user preferences are stored locally using `localStorage`.

  Modern Dashboard U
  - Neon cyber-inspired design with responsive layout and smooth interactions.

  Streak Badges
  - To encourgae users to continue working hard

  Downloadable file 
  -Export workout history as a downloadable file so users can keep track of their progress


 What This Project Demonstrates
 This project highlights practical front-end skills, including:
- DOM manipulation with JavaScript
- Event-driven UI logic
- State management using `localStorage`
- Responsive dashboard layouts
- Basic data visualization using Chart.js
